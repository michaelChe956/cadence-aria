**Goal:** 将 `src/product/models.rs`、`src/product/coding_models.rs`、`src/web/workspace_ws_types.rs` 三个纯数据模型文件拆分为目录模块，每个子文件不超过 800 行，保持外部 API 不变，所有测试通过。

**Architecture:** 采用 Rust 目录模块 + Facade 模式。原文件变为 `xxx/mod.rs`，内部按业务域拆分为多个子模块，`mod.rs` 统一 `pub use` 重新导出公共类型。外部 `use` 路径和公共符号保持不变。

**Tech Stack:** Rust 2024, Cargo, rust-toolchain.toml 固定 1.95.0
