# Aria Web 工作台 P3 前端基础 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 搭建 React/Vite/TypeScript 前端基础，提供 API client、router、workbench store、新建任务和继续任务基础入口。

**Architecture:** P3 只建立前端基础壳和数据访问层，保持第一屏为工作台，不做 landing page。API 类型必须镜像 P1/P2 的 snake_case JSON。

**Tech Stack:** pnpm、React、Vite、TypeScript、TanStack Router、Tailwind CSS、Vitest、Testing Library、lucide-react。

---

## Design Coverage

P3 覆盖：

- React/Vite/TypeScript/TanStack Router/Tailwind 技术选型
- 第一屏即工作台
- 新建任务输入：request、change-id、policy preset、provider mode、timeout
- 继续已有 task 的 API client、router、store 基础结构；可视化 task selector 在 P4 完成
- URL search params 恢复节点、tab、artifact、turn 的基础结构
