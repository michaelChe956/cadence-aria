# Aria 本地正式环境全流程调试 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 macOS Intel 与 Win11 WSL Ubuntu 上完成 Aria 本地正式环境全流程调试，并用 `naruto` 测试 worktree 验证真实 Claude Code、Codex、superpowers、OpenSpec 驱动的 JWT 登录功能闭环。

**Architecture:** 采用分层执行：先验证本机环境与 Aria 自身，再验证 daemon/REPL、fake provider 链路、真实 provider preflight，最后执行 `naruto` worktree 的正式 E2E。外部只审计 Aria 产物，不手工替 Aria 开发或测试 `naruto`。

**Tech Stack:** Rust 1.95.0、Cargo、Docker `rust:1-bookworm`、Claude Code CLI、Codex CLI、OpenSpec、superpowers、Git worktree、Node.js、pnpm、Vite React、Express、JWT。
