# 横切能力文档：artifact_validate

## 1. 能力标识

- 能力 ID：`CC04`
- 能力名称：`artifact_validate`
- 类型：校验
- 适用范围：全部产生产物的节点

## 2. 能力目的

保证节点不会因为只生成了文本而被误判完成，所有产物都要通过最小结构校验。

## 3. 触发条件

- 节点声明已生成主产物
- 交接前

## 4. 前置状态与输入

- artifact ref
- artifact type
- 对应产物规范

## 5. Aria 执行动作

1. 读取产物
2. 对照产物规范校验最小字段
3. 写入 validation record
4. 校验失败时阻止交接

## 6. 状态变化与副作用

- 生成 validation record
- 成功则允许交接
- 失败则回流或挂 gate

## 7. 输出与记录

- validation record
- artifact validity summary

## 8. 完成判定

校验记录写入成功且结果已反馈给原节点。

## 9. 失败与恢复

- 校验逻辑失败：视为节点失败
- 产物缺失：直接回流到产出节点

## 10. 与节点文档的关联规则

所有节点文档必须声明哪些产物要经过该能力校验。
