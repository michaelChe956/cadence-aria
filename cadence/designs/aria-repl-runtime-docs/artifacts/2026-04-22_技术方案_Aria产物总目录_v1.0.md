## 说明

> **斜杠节点语义**：部分产物的"默认产出节点"包含斜杠（如 `N16/N19`），表示该产物可在多个节点产出。斜杠前为主产出节点，斜杠后为补充产出节点（通常为 rework 后的重新产出）。

> **runtime_snapshot**：所有 Aria 内部节点产出的 runtime_snapshot 由全局协议 §16 统一定义，不在此表中单独列出。

> **格式规则**：用户可读文档使用 Markdown 格式，机器消费的结构化记录使用 JSON 格式。
