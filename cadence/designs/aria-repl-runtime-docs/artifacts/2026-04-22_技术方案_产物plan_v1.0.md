# 产物规范：plan

## 1. 产物标识
- 类型 ID：`ART-PLAN`
- 类别：文档产物
- 默认节点：`N11`

## 2. 产物目的
定义实施工作包、依赖、验收与并行提示，是 dispatch 的直接输入。

## 3. 产出时机
readiness 通过后产出。

## 4. 产物存储位置
默认写入 `cadence/plans/`。

## 5. 最小字段清单
| 字段 | 必填 |
|------|------|
| `work_packages` | 是 |
| `dependencies` | 是 |
| `acceptance_checks` | 是 |
| `parallelism_hints` | 是 |

## 6. 推荐结构
工作包、依赖、验收、并行说明。

## 7. 固定格式示例
```md
# Plan
## Work Packages
## Dependencies
## Acceptance Checks
## Parallelism Hints
```

## 8. 校验规则
至少 1 个 work package 和 1 组 acceptance checks。

## 9. 交接规则
供 `N12` 消费。

## 10. 版本与修订规则
方案变化时升级版本。

## 11. 失败与缺失处理
缺 work package 则回流 `N11`。
